
# 🪙 CoinFlow — Micro-Tasking & Earning Platform

A full-stack MERN micro-tasking platform where Workers earn coins completing tasks, Buyers post tasks, and Admins manage the platform.



## ✨ Features
- Three-role system — Worker, Buyer, Admin with protected dashboards
- Coin economy — complete tasks to earn coins, 20 coins = $1
- Stripe payment integration for buying coins
- Real-time notification system for task actions
- imgBB image upload for profiles and tasks
- JWT authentication with persistent private routes
- Google OAuth login support
- Task review workflow with approve/reject actions
- Withdrawal system with admin approval
- Paginated submission history
- Fully responsive including dashboard sidebar

## 🛠️ Tech Stack
**Frontend:** React, Vite, React Router, TanStack Query, Firebase, Stripe.js, Swiper  
**Backend:** Node.js, Express, MongoDB, JWT, Stripe
